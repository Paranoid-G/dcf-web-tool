## 本次更新

### 計算核心全面升級

| 功能 | 說明 |
|-----|------|
| **子女教育支出** | 根據子女出生日期計算年齡，自動判斷學段，使用對應教育費用 |
| **貸款還款** | 計算退休前的總還款金額 |
| **大額支出** | 支持現值/終值，按支出年份計入 |
| **醫療通脹** | 退休期醫療支出使用獨立通脹率 |
| **教育通脹** | 教育支出使用獨立通脹率 |

### 使用的參數（完整版）

**已使用（16個）**：基本資料、退休計劃、退休金來源、整體通脹  
**新增使用（22個）**：
- 子女資訊：`child_count`, `child_birth_${i}`
- 教育費用：`edu_primary`, `edu_middle`, `edu_high`, `edu_college`, `edu_master`, `edu_phd`
- 貸款資訊：`loan_count`, `loan_type_${i}`, `loan_payment_${i}`, `loan_years_${i}`
- 大額支出：`exp_name_${id}`, `exp_amount_${id}`, `exp_type_${id}`, `exp_year_${id}`
- 通脹細分：`inflation_edu`, `inflation_medical`
